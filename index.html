About Project

PRA is an autonomous survey analysis tool that will help your organization connect quickly to cover every source of feedback and analyze what is driving feedback volume, sentiment, effort or satisfaction of customers.

PRA will help businesses to detect sentiments in online conversations, helping them to understand how customers feel about their brand, product or service.

so how does the Public Relations Analyzer works?

PR Analyzer will help saving time as well as resources by automating current tedious feedback managing process using various machine learning algorithms and Natural Language processing.

We are using different machine learning algorithms to achieve higher accuracy for text analysis processes such as sentiment analysis, topic modeling, keyword extraction, entity recognition etc. to draw a conclusion from large chunk of textual data.

Features

PRA is designed to simplify the process of feedback management by using Natural Language Processing and Deep Learning algorithms.

Data Collection

We are using Twitter/ Facebook APIs for collecting data to perform text analysis and store into high performing database.

Real Time Analytics

PRA collects data in real time as soon as it becomes available to get insights and draw conclusion immediately.

Topic modeling

It is a process to automatically identify topics present in a text object and to derive hidden patterns from a text corpus.

Data visualization
